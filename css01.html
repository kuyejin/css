스타일 기본 문법

문법: 선택자 { 속성1:값1; 속성:값2; }

선택자(Selector) : 서식을 지정하고 싶은 요소

속성(Properties) : 부여하고 싶은 스타일의 단어의 특성을 지정

값(Value) : 해당 속성으로 지정 가능한 키워드

[ 스타일 지정 방식 ]

인라인 스타일 방식

해당 요소에 style 속성(attibute)로 직접 지정하므로 스타일 지정 즉시 해당 스타일이 그 요소에만 지정된다.
재 사용성이 떨어지고, 변경하기가 힘듦.
그러므로, 가급적이면 사용하지 않도록 함

<span style="color: red;> 첫 단어 </span> 첫 단어
두 번째 단어
세번째 단어

내부 스타일 방식

해당 문서의 head태그 안 쪽에 style태그로 지정하여 현재 문서에만 적용할 경우 활용
재사용성은 현재 문서에만 적용되므로 중간 정도, 변경하기가 쉬움

<style> .commont { color:red } <style>

외부 스타일 방식

별도의 CSS 파일을 작성하여 해당 hrml문서에 연결하여 사용하는 방식으로 재사용성이 뛰어나고
필요한 곳에 link 태그를 활용하여 연결하고, 필요가 없는 경우 해당 link태그를 삭제하면 되고,
반드시 charset을 UTF-8로 처리하여야 한글이나 특수문자가 손상되지 않는다.

<link href="style.css" rel="stylesheet"/>
---------------------style1.css-------------------------
@charset "UTF-8"
선택자{속성:값;}

크기 단위

* 스타일에서 콘텐츠나 요소의 크기를 지정할 때 사용
em(rem)
% : 글자에는 안씀, 부모의 상대적인 크기
pt : 일반적인 글자 크기
px : 가장 많이 쓰임(글자크기, 박스크기), 모니터의 해상도에 따라 달리 보일 수 있음
vh: 모니터의 높이만큼 알아서 잡힘(반응형)
vw: 모니터의 폭에 맞춰서 알아서 잡힘(반응형)


색상 단위

* 웹 컬러를 나타내는 방식
컬러명, 16진코드 6자리, 16진 코드 3자리.
hsl(색각도(0~360),채도,명도), hsla(색각도(0~360),채도,명도,불투명도),
rgb(red, green, blue), rgba(red, green, blue, alpha)


각도 단위

* 해당 요소를 회전하려고 할 경우 활용
deg(각도), turn(바퀴)


시간 단위

* 애니메이션이나 지연시간 등에 지정함
s(초), ms(밀리초)
