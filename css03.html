css 출력 및 박스 관련 속성

* 통합 속성을 건 후에 세부 속성을 걸어야 한다.

* 속성 종류: display, cisibility, opacity, width, height, overflow, margin, padding, border 등

* 모든 속성의 값에 initial(초기값), inherit(부모요소로 부터 상속)

display (출력속성)

* 화면 출력 모드를 지정하는 속성

* 태그마다 화면 출력 모드가 block, inline, inline-block 등으로 구성됨

inline 속성 요소: span, a, strong, em, ins 등
lnline 요소의 특징 : 한 줄 안에 자동 줄바꿈이 될 때까지 배치되며, 크기 지정을 할 수 없어
width, height, line-height, text-indent, text-align, padding-top, padding-botton, margin-top, magin-bottom 등의
속성을 지정한다고 하더라도 결과에 반영되지 못한다.

block 속성 요소:div, h1~h6, p, section, article, header, footer 자동으로 다음 줄에 배치가 되며, 크기와 관련된 속성을 지정할 수 있음

부여 가능한 속성의 값: inline | block | inline-block | none

부여 가능한 속성의 값2: contents, flex, grid, box, table, inline-flex, inline-grid, inline-table, list-item, run-in,
table-cation, table-column, table-row, table-cell, table-header-group, table-footer-group,
table-column-group, table-row-group 등

visibility (가시속성)

* 보일지 보이지 않을지를 나타내는 속성

* visible(보이지 않고 영역도 차지하지 않음) | hidden(보이지 않지만 영역차지) | collapse

opacity(불투명도 속성)

* 불투명 정도를 나타내는 속성

* 0~1 까지의 실수 또는 정수로 표시 함

* 0-> 투명 | 1 -> 불투명

* 해당 요소의 글자와 배경색/ 배경이미지를 포함한 모든 콘텐츠를 불투명 또는 투명하게 지정할 수 있음

width (너비 속성)

* 가로(폭 = 너비)의 크기를 지정하는 속성

* auto | length | %

length : px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

* 기본 화면 출력 속성이 블록 요소 또는 인라인 블록요소 등에만 크기 지정이 가능함

height (높이 속성)

* 세로(높이)의 크기를 지정하는 속성

* auto | length | %

- length : px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

* 기본 화면 출력 속성이 블록 요소 또는 인라인 블록요소 등에만 크기 지정이 가능함

overflow(흘러넘침 속성)

* 콘텐츠의 양이 지정한 크기(폭/높이)보다 더 클 경우 지정

- auto(콘텐츠양이 크거나 많을 때만 스크롤 바를 표시)

- visible(흘러넘쳐도 표시해라)

- hidden(흘러넘치면 표시xxx)

- scroll(흘러넘치든 말든 스크롤바를 만들어라)

- clip

세부속성

overflow-x(가로), overflow-y(세로)

margin(바깥 여백)

* 현재 테두리(콘텐츠) 영역의 바깥 여백으로서 여러 콘텐츠와 콘텐츠 사이의 여백

* auto | length | %

- length : px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

margin: 값, margin:위/아래 좌/우, magin: 상 우 하 좌

세부속성

margin-top, margin-botton, margin-left, margin-right

* 주의사항: 통합 속성을 먼저 지정하고, 세부 속성을 나중에 지정하여야 세부 속성이 적용됨

padding(안쪽여백)

* 테투리와 콘텐츠 사이의 여백

* auto | length | %

- length : px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

padding:값, padding:위/아래 좌/우, padding: 상 우 하 좌

세부 속성

padding-top, padding-botton, padding-left, padding-right

* 주의사항: 통합 속성을 먼저 지정하고, 세부 속성을 나중에 지정하여야 세부 속성이 적용되며,
지정한 폭/높이 padding과 border값을 더한 값이 실제 배치(적용)의 크기가 된다.

border(테두리)

* bordr: width style color

ex) border: 1px solid #000; -> 1px 두께의 검은색 실선

- width(선의 두께): px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

- style(선의 모양): solid, none, hidden, dotted, dashed(사각점선), double, groove(음각), ridge(양각), inset, outset

- color(선의 색): 컬러명, 16진(hex)코드, rgb()/rgba(),hsl(,hsla()등으로 지정이 가능

카테고리별 세부속성

border-width(선의 두께) - 상 우 하 좌

ex1) border-width:2px

ex2) border-width:2px 5px

ex3) border-width:2px 5px 10px 20px

border-style(선의 종류) - 상 우 하 좌

ex1) border-style: solid

ex2) border-style:solid dashed

ex3) border-style:solid dashed double hidden

border-color(선의 색깔) - 상 우 하 좌

ex1) border-color:red

ex2) border-color:red blud

ex3) border-color:red blud gold orange

`

방향 별 세부 속성

border-left(왼쪽 테두리)

ex1) border-left: 2px double silver

border-right(오른쪽 테두리)

ex2) border-right: 5px dotted gold

border-top(위쪽 테두리)

ex3) border-top: 4px dashed deepskyblue

border-bottom(아래쪽 테두리)

ex4) border-bottom: 7px ridge deeppink

세세속성

border-left-width, border-left-style, border-left-color

border-right-width, border-right-style, border-right-color

border-top-width, border-top-style, border-top-color

border-bottom-width, border-bottom-style, border-bottom-color

각 방향별로 선두께, 선종류, 선색상을 개별적으로 지정할 수 있음

box-sizing(박스의 크기 산정 방법)

* 박스의 크기를 지정한 너비/ 높이가 콘텐츠까지만 할지, padding과 border를 포함하여 적용할 지 등의 여부를 지정

* content-box | border-box

- content-box : 지정한 너비/높이가 콘텐츠에만 적용

- border-box : 지정한 너비/높이가 콘텐츠의 + padding + border로 적용됨

border-radius(테두리 둥글기)

* px, pt, pc, cm, mm in, vw(폭), vmin(현재 기기의 가장 좁은 폭), vmax(현재 기기의 가장 넓은 폭) 등의 단위로 지정이 가능

ex1) border-radius: 20px;

ex2) border-radius: 50%;

ex3) border-radius: 20px 50%;

ex4) border-radius: 20px 50% 30px 40px; (상 우 하 좌)

ex5) border-top-left-radius: 20px;

ex6) border-top-right-radius: 20px;

ex7) border-bottom-left-radius: 20px;

ex8) border-bottom-right-radius: 20px;
